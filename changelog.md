# Журнал изменений

## Версия 0.4.0 (2023-06-20)

### Улучшения структуры
- **Реорганизация кода**: Выделен отдельный файл `bot.ts` для инициализации бота и `routes.ts` для маршрутизации
- **Типы данных**: Вынесены типы и интерфейсы в отдельные файлы в директории `models`
- **Разделение ответственности**: Прямые обработчики перенесены из `index.ts` в соответствующие контроллеры
- **Обработка ошибок**: Реализована централизованная система обработки ошибок в `utils/errorHandler.ts`
- **Сбор метрик**: Добавлен `metricsMiddleware` для отслеживания производительности запросов

### Исправления ошибок
- **Исправлен баг с правами доступа**: Удалено условие `ctx.message.text?.startsWith('/')` из `adminMiddleware`, которое неправильно позволяло всем командам обходить проверку прав администратора
- **Доработаны тесты**: Исправлены импорты в тестовых файлах для корректной работы с Bun
- **Регистрация контроллеров**: Подключены все контроллеры, включая ранее пропущенный `ratingController`

### Улучшения кода
- **Утилитарные функции**: Добавлены вспомогательные функции для проверок в middleware
- **Константы**: Вынесены строковые константы для предотвращения "магических строк"
- **Типизация**: Усилена типизация в сервисах и контроллерах
- **Middleware**: Улучшена структура middleware и порядок их применения

### Дополнительные изменения
- **Документация**: Добавлены JSDoc комментарии к ключевым функциям
- **Порядок инициализации**: Исправлен порядок инициализации сервисов и компонентов
- **Оптимизация импортов**: Устранены потенциальные циклические зависимости

## Версия 0.3.1 (2025-04-13)

- Добавлено подробное логирование для отладки выполнения команд
- Добавлен безопасный метод сериализации данных с корректной обработкой BigInt

- Решены проблемы с обработкой команд /profile, /viewprofile и /members
